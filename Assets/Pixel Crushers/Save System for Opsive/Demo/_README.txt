Save System for Opsive Controllers


Before playing the demo, add DemoScene and DemoLoadingScreen to the project's
build settings. 

If you haven't already enabled Pixel Crushers support for the Input System
package, select menu item Tools > Pixel Crushers > Common > Misc > Use New
Input System....

This scene uses prefabs from Opsive's URP Demo scene. Make sure you've set up
your project for URP.
